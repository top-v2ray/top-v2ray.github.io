什么是Shadowrocket？

Shadowrocket（小火箭）是一款专为iOS平台设计的网络代理工具，支持多种常见的代理协议，如V2Ray、Shadowsocks等。通过配置代理服务器，Shadowrocket能够帮助用户绕过网络限制，保护上网隐私，提供更安全、更高效的互联网体验。其强大的功能和用户友好的界面，使其成为广受欢迎的工具之一。

小火箭Shadowrocket的主要功能

1. 多种代理协议支持

小火箭支持多种主流的代理协议，包括V2Ray、Shadowsocks等。无论你是使用Shadowsocks作为日常上网工具，还是需要V2Ray提供更高的加密性和隐私保护，Shadowrocket都能满足你的需求。通过选择合适的协议，用户可以根据自身的网络环境和安全需求来灵活配置代理。

2. 简单易用的界面

Shadowrocket的界面设计直观、清晰，特别适合新手使用。所有配置选项都清楚地列在主界面，用户可以轻松找到所需功能。配置代理服务器时，输入相关信息也非常简便，不需要复杂的操作流程，即使是没有技术背景的用户也能快速掌握。

3. 流量管理与监控

除了提供代理服务外，Shadowrocket还具备流量管理功能。用户可以实时查看自己的网络流量使用情况，帮助监控网络资源的消耗。对于那些有数据流量限制的用户，或者想要避免不必要消耗的用户来说，流量管理无疑是一个非常实用的功能。

4. 自动分拨功能

Shadowrocket具有自动分拨功能，可以根据不同网站和资源的访问需求自动选择最优的代理通道。该功能能够确保用户在访问受限网站时获得最佳速度和稳定性，同时还能够有效避免因节点问题导致的网络中断。

如何在iOS设备上安装Shadowrocket？

安装Shadowrocket是使用该工具的第一步，下面是详细的安装步骤：

步骤一：下载与安装

  1. 打开App Store：首先，打开您的iPhone或iPad上的App Store。

  2. 搜索Shadowrocket：在搜索框中输入“Shadowrocket”，找到该应用程序。

  3. 下载并安装：点击“获取”按钮，下载并安装应用程序。由于Shadowrocket是一款付费应用，您需要进行购买后才能下载。

  4. 启动应用：安装完成后，点击“打开”按钮启动应用程序。

步骤二：确保iOS设备已更新

在安装之前，确保您的设备已更新到最新版本的iOS操作系统。小火箭的某些功能可能在较老的iOS版本中无法正常工作，因此保持操作系统的更新至关重要。

步骤三：切换App Store地区（如适用）

在某些地区，可能会由于App Store的区域限制导致无法直接下载Shadowrocket。此时，您可以尝试更改App Store的地区或使用其他方式获取该应用。

配置Shadowrocket的步骤

安装完Shadowrocket后，您需要配置代理服务器，才能使其正常工作。下面是配置代理的详细步骤。

步骤一：添加代理服务器

  1. 打开Shadowrocket应用：点击应用图标进入Shadowrocket界面。

  2. 进入配置页面：在主界面上，点击右上角的“+”按钮，进入配置选项页面。

  3. 选择代理类型：在弹出的菜单中，选择代理类型（例如Shadowsocks或V2Ray）。这些信息通常由您的代理服务商提供。

  4. 输入代理信息：输入您所获得的代理服务器的地址、端口、密码、加密方式等信息。确保所有信息无误。

  5. 保存配置：完成输入后，点击右上角的“保存”按钮。

步骤二：测试代理设置

配置完成后，您可以通过“测试”功能检查代理设置是否正确。点击配置页面中的“测试”按钮，Shadowrocket会自动验证设置的有效性。如果代理服务器信息正确无误，您会看到“成功”的提示。

步骤三：连接代理

一旦配置完毕，返回到主界面，点击您刚才添加的代理配置，然后选择“连接”。如果一切正常，连接状态将显示为“已连接”，您就可以开始使用代理服务了。

FAQ

Netch 全局代理与规则代理的区别是什么？
全局代理会将所有流量通过节点转发，适合游戏或全局科学上网；规则代理只针对指定应用或 IP 走节点，优化带宽利用和网络效率，根据不同需求选择使用模式。
VLESS 在高封锁环境下的优势是什么？
VLESS 去掉了内置加密，依赖 TLS 层安全，减少被 DPI 检测的特征。结合 XTLS 或 Reality 模式，可以更好地隐藏流量，提高在高封锁网络中的稳定性与隐蔽性。
Mellow 的 TUN 模式与 SOCKS5 代理区别是什么？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响特定应用。TUN 更适合全局加速、游戏和视频应用，保证所有流量走节点，提高网络稳定性和速度。
如何将 Windows 上的 V2Ray 与 WSL 环境联合调试？
在 WSL 内运行需要的调试工具并通过 localhost 或将 V2Ray 监听地址绑定为 0.0.0.0，然后在 Windows 与 WSL 之间配置端口映射。注意 Windows 防火墙规则需允许 WSL 与 Windows 进程通信。
如何把 Clash 规则与本地 hosts 文件结合使用？
可在 Clash 的 rules 中优先设置域名直连规则，并在系统 hosts 中添加常用站点的固定解析以加速 DNS 解析。注意 hosts 优先级高于 DNS，因此修改 hosts 需谨慎以避免与真实解析冲突。
Sing-box 可以替换 Clash 配置文件吗？
Sing-box 支持直接解析 Clash YAML 配置，并转换为自身 JSON 格式。节点、策略组和规则基本兼容，但部分自定义脚本可能需要手动调整才能完全使用。
Trojan-Go 的多路复用（mux）如何开启？
Trojan-Go 在配置文件中通过启用 mux 参数即可实现多路复用。此功能允许多个连接共享同一隧道，减少握手延迟并提升整体传输效率，尤其在移动网络中表现出色。
Sing-box 的策略组支持按延迟选择节点吗？
支持。Sing-box 会根据节点延迟自动选择最优节点，结合策略组和路由规则，实现智能分流。延迟最低的节点优先被使用，提高网络访问速度和稳定性。
Vmess 节点配置时常见错误有哪些？
常见错误包括端口号不匹配、UUID 错误、TLS 设置不正确或传输类型配置错误。这些都会导致连接失败。检查服务器配置与客户端参数一致性是解决问题的关键。
Hiddify Next 用户带宽限制如何配置？
管理员可设置每个用户上传、下载和总流量上限，超出限制可自动降速或阻止访问。保证服务器资源合理分配，避免单用户占用过多带宽，提高整体网络公平性。
